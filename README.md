# TikTokDemo
仿抖音录制、特效处理、裁剪视频（也与微信裁剪视频类似）

####仿抖音的一个Demo，包含美颜、滤镜、BGM裁剪、抖动特效和倒放等，以及裁剪视频等。

####仅供参考。

由于原本项目中功能比较复杂，只整理了部分，以及去除了不必要的功能。
没有用到原项目中的架构，只是做了简单整理，所以代码可能会比较糙。
